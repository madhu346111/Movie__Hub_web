# Movie__Hub_web
🎬 Movie Hub Pro

A modern and responsive Movie Hub Web Application built using HTML, CSS, and JavaScript.
Users can explore movies, watch trailers, view details, rate movies, and manage a personal watchlist.

🚀 Features
🎥 Movie Features
▶️ Watch movie trailers (YouTube embedded player)
📄 View detailed movie information
⭐ Interactive rating system (5-star rating)
❤️ Add/Remove movies from Watchlist (saved in LocalStorage)
🔍 Filtering & Search
🔎 Search movies by title
🎭 Filter movies by genre:
Hollywood (Action)
Bollywood (Drama)
Cartoon (Animation)
🔥 UI Features
🔥 Trending movies section (horizontal scroll)
📱 Fully responsive (mobile + desktop)
🎨 Modern gradient UI design
🧾 Footer with links and newsletter
